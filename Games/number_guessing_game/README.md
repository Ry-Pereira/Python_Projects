# 🎯 Number Guessing Game 🐍

## 📌 Overview

This is a Python-based Number Guessing Game where the player tries to guess a randomly selected number within a chosen range 🎲.

The game includes two modes:
- ⏳ Limited lives mode
- ♾️ Unlimited lives mode

The player receives feedback after each guess to help them find the correct number.

---

### 🎯 Goals of the Project

* 🧠 Practice Python fundamentals  
* 🎮 Build a simple interactive game  
* 🔁 Use loops and conditional logic effectively  
* 🎲 Work with random number generation  
* ⌨️ Handle user input in real-time  

---

## 🧱 Project Structure 📁

This project consists of a single Python file:

* 📄 `number_guessing_game.py`  
  - Generates a random number  
  - Takes user input for guesses  
  - Provides hints (too high / too low)  
  - Tracks lives (if enabled)  
  - Determines win or loss outcome  

---

## 🛠️ Technologies Used 💻

* 🐍 Python 3.x  
* 🎲 `random` module  
* ⌨️ Input handling (`input()`)  
* 🔁 Loops (`while`, `for`)  
* ⚙️ Conditional statements (`if`, `else`)  

---

## ▶️ How to Run

1. 📁 Open the project folder  
2. ▶️ Run the Python file:

```bash
python number_guessing_game.py
```

3. 🎮 Follow the prompts in the terminal:
   - Choose range
   - Select lives mode
   - Start guessing!

---

## 🎮 Game Features

* 🔢 Custom number range selection  
* ❤️ Optional limited lives system  
* ♾️ Unlimited guessing mode  
* 📢 Feedback after each guess  
  - “Too big”  
  - “Too small”  
  - “Correct”
