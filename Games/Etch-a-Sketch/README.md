# 🎨 Etch-A-Sketch Python Program 🐢

## 📌 Overview

This repository contains a simple Python turtle graphics project that recreates a digital Etch-A-Sketch experience 🎮.

The program allows users to control a turtle using keyboard inputs to draw shapes and patterns interactively on the screen ✏️🐢.

---

### 🎯 Goals of this Project

* 🧹 Practice Python programming fundamentals  
* ⌨️ Learn keyboard event handling  
* 🐢 Explore turtle graphics in Python  
* 🎨 Create an interactive drawing application  

---

## 🧱 Project Structure 📁

This project contains:

* 🐢 A Turtle object for drawing  
* 🖥️ A Screen object for displaying graphics  
* ⚙️ Functions for movement and controls  
* ⌨️ Keyboard bindings for user interaction  

The file included in this project is:

* `Etch-A-Sketch.py` — Main Python program file  

---

## 🛠️ Technologies Used 💻

* 🐍 Python 3.x  
* 🐢 Turtle Graphics Module  

---

## 📖 Usage Instructions ▶️

To run the project:

1. 📁 Navigate to the project folder  
2. ▶️ Run the Python file:

```bash
python Etch-A-Sketch.py
```

3. 🎮 Use the keyboard controls to draw on the screen
