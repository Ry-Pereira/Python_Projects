Project for Hangman Program

A simple Hangman game built in Python using **Tkinter**.  
The player tries to guess a randomly selected word by selecting letters. Wrong guesses gradually draw the hangman, and the game ends when the player either guesses the word or runs out of lives.

## Features

- Graphical interface using Tkinter.
- Randomly selects a word from a predefined list.
- Displays the word with blanks that get filled in as the player guesses correctly.
- 6 lives system with hangman stages represented in ASCII art.
- Buttons for each letter of the alphabet to make guesses.
- Win and lose screens with the option to restart or quit.

## Installation

1. Make sure Python 3.x is installed on your system.
2. Install Tkinter if it is not already installed (usually comes pre-installed with Python).
