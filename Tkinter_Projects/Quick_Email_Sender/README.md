📂 Project Name

Quick Email Sender

📝 Description

Quick Email Sender is a simple desktop application built using Python and Tkinter that allows users to compose and send emails quickly through a graphical user interface (GUI). The app provides fields for entering the recipient's email address, subject, and message, along with basic validation and user prompts.

🚀 Features

User-friendly GUI built with Tkinter

Input fields for:

Recipient email ("To")

Subject line

Message body

Input validation with pop-up alerts

Option to send emails with empty subject or message (with confirmation)

Clear all fields with one click

Exit button to close the application

Draft-saving feature placeholder (to be implemented)

🛠️ Technologies Used

Python

Tkinter (for GUI)

smtplib (for email functionality)
