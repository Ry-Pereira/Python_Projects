# ⏱️⌛🕒 Simple Timer App 🕒⌛⏱️

## 📌 Description

This project is a graphical countdown timer application built using **Python** and **Tkinter** 🐍🖥️.  
The app allows users to enter a countdown time in **MM:SS format**, start the timer ▶️, pause or resume it ⏸️▶️, and restart the timer 🔄 whenever needed.

A popup alert 🚨 appears when the timer reaches `00:00`.

---

## ✨🚀 Features

✅ Input time in **MM:SS format**  
✅ Start countdown timer ▶️  
✅ Pause and resume timer ⏸️▶️  
✅ Restart timer anytime 🔄  
✅ Popup alert when time is complete 🚨  
✅ Input validation for incorrect formats ❌  
✅ Clean and simple Tkinter GUI 🖥️  
✅ Beginner-friendly Python project 🐍  

---

## ⚙️🧠 How It Works

1️⃣ User enters a time such as `05:30` ⌨️  
2️⃣ Click **Start** ▶️ to begin countdown  
3️⃣ Use **Pause/Unpause** ⏸️▶️ to control the timer  
4️⃣ Click **Restart** 🔄 to reset the countdown  
5️⃣ A popup notification 🚨 appears when the timer ends
