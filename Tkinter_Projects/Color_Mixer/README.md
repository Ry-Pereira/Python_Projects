# 🎨🖌️ Color Mixer Application 🖌️🎨

## 📌 Description

This project is a graphical **Color Mixer Application** built using **Python** and **Tkinter** 🐍🖥️.  
The program allows users to mix colors interactively by clicking color buttons. Each selected color is blended into the current mix, and the resulting mixed color is displayed live on the canvas.

Users can also reset the color mix and start over at any time 🔄.

---

## ✨🚀 Features

✅ Interactive Tkinter GUI 🖥️  
✅ Mix multiple colors together 🎨  
✅ Live color blending on canvas 🌈  
✅ RGB color averaging logic 🧠  
✅ Reset color mix anytime 🔄  
✅ Uses RGB and HEX color conversion 🔢  
✅ Beginner-friendly Python GUI project 🐍  

---

## ⚙️🧠 How It Works

1️⃣ User clicks a color button 🎨  

2️⃣ The selected RGB color is added to the mix ➕  

3️⃣ The application calculates the average RGB values 🧮  

4️⃣ The mixed RGB color is converted into HEX format 🔢  

5️⃣ The canvas background updates to display the new mixed color 🌈  

6️⃣ User can reset the mix anytime using the reset button 🔄

## 🧠 RGB to HEX Conversion

The program converts RGB tuples into HEX color values using:

```python
convert_rgb_to_hex_color(rgb_color_value)
```

Example:

```python
(255, 0, 0) → #FF0000
```
