Project for Tic-Tac-Toe Program

A simple graphical Tic-Tac-Toe game built in Python using **Tkinter**.  
Two players can play against each other on the same computer. The game automatically tracks turns, detects wins, ties, and allows restarting or quitting.

## Features

- Graphical interface using Tkinter.
- Randomly chooses which player (X or O) starts first.
- Highlights the current player's turn.
- Detects win or tie conditions automatically.
- Buttons for restarting or quitting the game after it ends.

## Installation

1. Make sure you have Python installed (version 3.x recommended).  
2. Install Tkinter (usually comes pre-installed with Python).
