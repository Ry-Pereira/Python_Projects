# 🎤💬 Kanye West Quote Generator 💬🎤

## 📌 Description

This project is a graphical **Kanye West Quote Generator** built using **Python**, **Tkinter**, and an external API 🐍🖥️🌐.  
The application fetches random Kanye West quotes from an online API and displays them inside colorful speech bubbles.

Users can click on the Kanye West image to generate a new random quote 🎲.

---

## ✨🚀 Features

✅ Fetch random Kanye West quotes from an API 🌐  
✅ Interactive Tkinter GUI 🖥️  
✅ Random colorful speech bubbles 🌈  
✅ Clickable Kanye West image 🎤  
✅ Dynamic quote generation 🔄  
✅ API response handling ✅  
✅ Beginner-friendly Python project 🐍  

---

## ⚙️🧠 How It Works

1️⃣ User launches the application ▶️  

2️⃣ The GUI displays a Kanye West image 🎤  

3️⃣ User clicks the image 🖱️  

4️⃣ The program sends a request to the Kanye quote API 🌐  

5️⃣ A random quote is retrieved 🎲  

6️⃣ The quote appears inside a randomly colored speech bubble 💬🌈  

7️⃣ Clicking again generates a new quote 🔄
