Project for Kanye West Quote Generator

Project Description

A Tkinter-based Python application that fetches random quotes from Kanye West via an API. Every time the user clicks on the Kanye West image, a new quote appears inside a randomly colored speech bubble.

This project demonstrates:

Working with APIs using the requests module.
Creating interactive GUIs with Tkinter.
Using images and canvas widgets for a visually appealing interface.
Features
Display a Kanye West image that the user can click.
Show a random Kanye quote in a colorful speech bubble.
Change the speech bubble color randomly with each new quote.
Simple and intuitive user interface.
Handles API errors gracefully.
