# 🌐 Flask Projects Repository 🐍

## 📌 Overview

This repository contains a collection of Flask-based Python web projects 🌍💻.

These projects focus on building web applications using the Flask framework 🧪, including dynamic routing, backend logic, and API integration 🔌. Some projects also interact with external APIs to fetch and display real-time data 📡.

---

### 🎯 Goals of this Repository

* 🧹 Maintain clean and organized Flask project structure  
* 🌐 Build dynamic web applications using Python  
* 🔌 Learn and apply API integration techniques  
* 📈 Improve backend development skills  
* ⚙️ Practice routing, templates, and server-side logic  

---

## 🧱 Project Structure 📁

Each folder contains a separate Flask project.

📌 Included Projects:

* 🌈 Background Color Changer  
  - Changes webpage background color based on URL input  
  - Demonstrates routing and dynamic responses in Flask  
  - Simple and interactive web application  

* 🔌 API-Based Projects  
  - Fetch data from external APIs  
  - Display real-time information in the browser  
  - Practice working with JSON responses  

---

## 🛠️ Technologies Used 💻

* 🐍 Python 3.x

## 📖 Usage Instructions ▶️

Each Flask project is stored in its own folder 📂.

To run a project:

1. 📁 Navigate to the project folder  
2. 📦 Install Flask (if not installed):

```bash
pip install flask
```

3. ▶️ Run the application:

```bash
python server.py
```

4. 🌍 Open your browser and go to:

```
http://127.0.0.1:5000/
```
