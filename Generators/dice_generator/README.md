# 🎲🎮🔥 Dice Generator Program 🔥🎮🎲

## 📌 Overview

This project is a Python-based Dice Generator application that allows users to create and roll numbered dice through a simple command-line interface. Users can either generate a random-sided dice or manually choose the number of sides for the dice.

The program provides:
- Random dice generation  
- User-selected dice generation  
- Menu-driven interaction  
- Dice rolling functionality

## ▶️ How to Run the Project

1. Make sure Python is installed.

2. Run the application:

```bash
python main.py
```

3. Follow the menu prompts in the terminal.

---

## 🌐 Menu Options

| Option | Description |
|--------|-------------|
| 1 | Generate a random numbered dice |
| 2 | Choose a numbered dice |
| 3 | Exit program |

---

## 🎲 Program Functions

| Function | Description |
|----------|-------------|
| `intro_to_program()` | Displays the welcome message |
| `print_menu()` | Displays menu options |
| `menu_choice()` | Handles user menu selection |
| `roll()` | Rolls the dice |
| `generate_random_numbered_dice()` | Creates a random-sided dice |
| `choosing_numbered_dice()` | Creates a custom-sided dice |
| `main()` | Main program execution |
