# 🔐🛡️🔑 Password Generator Program 🔑🛡️🔐

## 📌 Overview

This project is a Python-based Password Generator application that creates randomized passwords using numbers, letters, and special characters. Users can customize how many numbers, letters, and special characters they want included in the generated password.

The program provides:
- Random password generation  
- Customizable password length and content  
- Randomized password character ordering  
- Simple command-line interaction

## ▶️ How to Run the Project

1. Make sure Python is installed.

2. Run the application:

```bash
python main.py
```

3. Follow the prompts in the terminal to generate a password.
